JSON

什么是JSON

  JSON（JavaScript Object Notation，JavaScript对象标记），是一种轻量级的数据交换格式（非常常用！）

特点

  • 完全独立于编程语言的文本格式（也就是字符串）来存储和表示数据
  • 简洁且层次清晰，所以是轻量级的
  • 易于人的阅读和编写，也易于机器的解析和生成，大大提高了数据在网络中传输的效率

语法格式：

  • 对象使用键值对表示，键值对之间使用逗号分隔
  • 花括号保存对象
  • 方括号保存数组
  • 字符串和布尔类型的属性值可以不用引号

JSON和JavaScript的关系

  JSON可以理解为表示JavaScript对象的字符串，使用文本的形式来表示JS对象

JSON和JavaScript对象的相互转换

  • JSON转换为JavaScript对象，使用JSON.parse()方法

  • JavaScript对象转换为JSON字符串，使用JSON.stringify()方法

Fastjson

  fastjson是阿里巴巴开发的专门用于将Java对象转换为JSON字符串的库，当然它也可以将JSON字符串转换为Java对象。

要使用fastjson首先需要添加maven依赖：

fastjson三个主要的类
==JSONObject==

  • JSONObject代表了json对象

  • JSONObject实现了Map接口

  • JSONObject代表json对象，可以通过getXXX()方法获取json对象中的数据，也可以用size()、isEmpty()等方法获取键值对的个数。本质上和操作Map相同

==JSONArray==

  • JSONArray代表json对象数组
  • JSONArray内部使用List实现

==JSON==

  • JSON用于JSONObject、JSONArray的转换
  • 主要实现了json对象、json对象数组、JavaBean对象、json字符串之间的相互转换

jQuery.ajax

jQuery中的ajax本质上就是封装了XMLHttpRequest，使得调用起来更加方便

部分参数如下：

  • url：请求地址
  • type：请求方式，GET、POST（1.9.0之后使用method）
  • headers：请求头
  • data：要发送的数据
  • contentType：即将发送信息至服务器的内容编码类型，默认值为：application/x-www-form-urlencoded; charset=UTF-8
  • async：是否异步
  • timeout：设置请求超时时间（毫秒）
  • beforeSend：发送请求前执行的回调函数（全局）
  • complete：完成之后执行的回调函数（全局）
  • success：成功之后执行的回调函数（全局）
  • error：失败之后执行的回调函数（全局）
  • accepts：通过请求头发送给服务器，告诉服务器当前客户端可以接收的数据类型
  • dataType：将服务器端返回的数据转换成指定类型
    • "xml": 返回 XML 文档，可用 jQuery 处理。
    • "html": 返回纯文本 HTML 信息；包含的 script 标签会在插入 dom 时执行。
    • "script": 返回纯文本 JavaScript 代码。不会自动缓存结果。除非设置了 “cache” 参数。注意：在远程请求时(不在同一个域下)，所有 POST 请求都将转为 GET 请求。（因为将使用 DOM 的 script标签来加载）
    • "json": 返回 JSON 数据 。
    • "jsonp": JSONP 格式。使用 JSONP 形式调用函数时，如 “myurl?callback=?” jQuery 将自动替换 ? 为正确的函数名，以执行回调函数。
    • "text": 返回纯文本字符串
